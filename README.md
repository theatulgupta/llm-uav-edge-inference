# NeuroEdge
### LLM-Guided Neural Network Compilation for Real-Time UAV Edge Inference

> M.Tech Thesis Project | PDPM IIITDM Jabalpur

NeuroEdge explores the use of **Large Language Models (LLMs)** as intelligent optimization agents for compiling and deploying deep neural networks on resource-constrained UAV edge platforms.

The goal is to automatically analyze neural network architectures, generate optimization strategies, and improve real-time inference performance for autonomous aerial systems.

---

## 🎯 Research Objectives

- Analyze neural network architectures using LLMs.
- Generate model-specific optimization strategies automatically.
- Optimize inference for resource-constrained UAV edge devices.
- Deploy optimized models on NVIDIA Jetson platforms.
- Benchmark LLM-guided optimizations against manual optimization techniques.
